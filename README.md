# 健康监测小程序 v3.0.0

微信小程序 - 心内科住院患者健康监测管理系统

## 功能特性

- 🏥 患者床位管理（支持出院/转科/恢复）
- 📊 健康指标监测（饮水量、尿量、体重、血压、心率、体温）
- 👩‍⚕️ 角色权限体系（护士、护工/家属、患者）
- 🔍 多维度筛选（性别、空床位、搜索）
- 📋 申请审核流程
- 🧑‍💼 护工/家属分配管理

## 技术栈

- 微信小程序原生开发
- 本地存储（wx.getStorageSync / wx.setStorageSync）

## 使用说明

1. 使用微信开发者工具导入本项目
2. 点击「生成演示数据」初始化 50 位患者数据
3. 护士账号：138****1111
